Types of functions

    

        There are two types of functions in JavaScript:
        
    
            
  1. Named functions
    2. 
            
    Functions which is declared with function keyword and have a indentifier name is called as 
                named function
            
    
                function demo(){
                    console.log("I am a named function")
                }
                
                demo();
            
    
            
    
            
    
            
  2. Anonymous functions
    3. 
            
    Functions which  does not have any identifier  name is called as 
                Anonymous function
            
    
                let fun=function(){
                    
                    console.log("I am a Anonymous function")
                }
                
                fun();
            
    
            
    
            
    

            
  3. Function with expression
    4.

                let a=function sum(a,b){
                    console.log(a+b);
